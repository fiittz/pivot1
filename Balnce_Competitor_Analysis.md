# Balnce Competitor Analysis & Pricing Strategy
**Date: March 2026**

---

## Executive Summary

Balnce is an all-in-one Irish accounting practice management platform. To replicate Balnce's functionality, an Irish accountant would need to subscribe to **5-7 separate tools** costing approximately **€700-€1,100/month per seat**. Based on the "10% of value created" model, Balnce should price at **€840-€1,320/year per seat** (or **€70-€110/month**).

---

## 1. Competitor Pricing Breakdown

### A. Core Accounting Software (needed for GL, TB, P&L, Balance Sheet)

| Product | Price | Model | Irish-Specific? |
|---------|-------|-------|----------------|
| **Xero** (Premium) | ~€38/mo (~£36/mo) | Per organisation, unlimited users | VAT returns (ROI), bank feeds, Hubdoc included |
| **QuickBooks Online** (Plus) | ~€45/mo (~£42/mo) | Per organisation + per user for payroll | VAT MTD, limited Irish features |
| **Sage Business Cloud** | ~€35-€50/mo | Per organisation | Irish VAT, some localisation |
| **FreeAgent** | ~€22/mo (~£19/mo) | Per organisation, unlimited users | UK-focused, limited Irish features |
| **Surf Accounts** | €15-€30/mo | Per organisation + €4/extra user | Irish-built, Irish chart of accounts |
| **Big Red Cloud** | €28/mo + VAT | Per organisation | Irish-built, Revenue-compliant |

**Typical cost for a practice:** Xero Partner or equivalent = ~€38-50/mo per client entity. A practice with 50 clients = **€1,900-€2,500/mo** just for accounting software.

### F. Payments (Invoice collection via Stripe)

| Product | Price | Model | Notes |

### G. eRCT (Construction Industry)

| Product | Price | Notes |
|---------|-------|-------|
| **No standalone SaaS competitor** | N/A | eRCT is typically handled via ROS manually or through expensive desktop packages |
| **ROS (Revenue)** | Free | Manual SOAP interface, extremely tedious |
| **Balnce** | Included | Full SOAP API integration: contract notifications, rate lookups, deduction authorisations, monthly returns |

**Value: Significant.** There is no cloud-based SaaS product in Ireland offering full eRCT integration. This is a major differentiator.

## 3. Pricing Recommendation

### Model: 10% of Value Created, Per Seat, Per Year

| Metric | Low | Mid | High |
|--------|-----|-----|------|
| Annual tool stack cost replaced | €12,516 | €16,650 | €20,784 |
| 10% of value created | €1,252 | €1,665 | €2,078 |
| **Monthly equivalent** | **€104/mo** | **€139/mo** | **€173/mo** |

### Recommended Pricing Tiers

| Tier | Price | Target |
|------|-------|--------|
| **Starter** | **€99/mo per seat** (€1,188/yr) | Solo practitioner, < 20 clients |
| **Professional** | **€149/mo per seat** (€1,788/yr) | Small practice, 20-75 clients |
| **Enterprise** | **€199/mo per seat** (€2,388/yr) | Multi-practitioner firm, 75+ clients, full eRCT, priority support |

### Why This Works

1. **At €149/mo**, Balnce costs less than Dext alone (€500-850/mo for 50 clients)
2. **One subscription** replaces 5-7 separate tools and their integrations
3. **No per-client or per-employee charges** -- a seat is a seat, regardless of client count
4. **eRCT is a moat** -- no cloud competitor offers this in Ireland
5. **Stripe Connect revenue** adds an additional income stream on top of seat pricing
6. **Irish-specific features** (PAYE cumulative, USC 4-band, PRSI Class A1/S, Directors PRSI, S.440 close company, Irish capital allowances) mean accountants don't need workarounds

## 4. Competitive Moats (Features No Single Competitor Matches)

| Feature | Nearest Competitor | Gap |
|---------|-------------------|-----|
| Full eRCT SOAP API integration | None (manual ROS only) | **No cloud competitor** |
| Irish payroll + accounting + practice mgmt in one | None | Requires 3+ tools |
| Cap table with S.440 close company detection | None (spreadsheets) | **Unique** |
| DWT calculation + dividend management | None (spreadsheets) | **Unique** |
| AI categorisation (suggest-only, never auto-applies) | Dext AI (auto-applies) | **Safer approach** |
| Stripe Connect on invoices (platform revenue) | Stripe direct (no platform cut) | **Revenue model advantage** |
| 12-column ETB export | TaxCalc Accounts Production | Built-in, not bolt-on |
| Client confirmation flow on working papers | None | **Unique** |
| Multi-period VAT (6 bi-monthly Irish periods) | Xero (limited), TaxCalc VAT Filer | Native, not bolt-on |
| 19-item onboarding checklist | AccountancyManager (partial) | More comprehensive |

---

## 5. Key Takeaways

1. **The Irish market is underserved.** Most tools are UK-first or US-first with Irish features bolted on. Balnce is Irish-first.

2. **The tool sprawl is real.** An Irish accountant today uses 5-7 subscriptions (Xero + BrightPay + Dext + AccountancyManager + TaxCalc + spreadsheets). Balnce consolidates all of this.

3. **€149/mo per seat is compelling.** It's roughly 10% of the value created and less than what most practices pay for Dext alone.

4. **eRCT is a land-and-expand wedge.** Construction-sector accountants have no cloud alternative. Win them with eRCT, expand to full platform adoption.

5. **Stripe Connect is recurring revenue on top of SaaS.** Every invoice paid through Balnce generates platform revenue, making the effective revenue per customer higher than the seat price alone.
